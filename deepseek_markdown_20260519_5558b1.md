# ✨ Рандомные пожелания

Сайт с генератором случайных тёплых пожеланий. Нажми на карточку или кнопку — получи порцию добра!

## 🌟 Особенности
- 🎲 50+ уникальных пожеланий
- 🔗 **Умные ссылки** — поделись пожеланием с другом
- 📊 Счётчик просмотров (сохраняется в браузере)
- 🎨 Красивый стекломорфный дизайн
- 📱 Адаптив под все устройства

## 🚀 Как использовать
1. Открой [сайт](https://ВАШ_ЛОГИН.github.io/random-wishes/)
2. Нажимай на карточку или кнопку "Новое пожелание"
3. Нажми "Скопировать ссылку" — отправь другу!
4. Друг увидит ТО ЖЕ САМОЕ пожелание

## 🛠 Технологии
- HTML5
- CSS3 (Flexbox, Grid, анимации)
- Vanilla JavaScript (без зависимостей)
